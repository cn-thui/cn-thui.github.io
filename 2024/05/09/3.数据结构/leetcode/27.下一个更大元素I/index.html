下一个更大元素I（496）

发表于 2024-05-09 分类于 数据结构与算法 ， leetcode

原题

  • 下一个更大元素I

Code

1.枚举

  • 算法思路：在第二个数组中查询第一个数组的元素，然后找到其后方第一个比该元素大的值，并按照要求插入；依次将第一个数组中的元素查找完毕即可。
  • 算法分析：双层循环，时间复杂度：O（n^2）；不考虑返回数组，使用常数空间，空间复杂度：O（1）。
  • 算法归纳：通过遍历两个数组，每次遍历元素都判定是否符合要求，属于枚举思想。
