最大连续1的个数

发表于 2024-05-08 更新于 2024-05-15 分类于 数据结构与算法 ， leetcode

原题

  • 最大连续1的个数

Code

  • 算法思路：如何得到最大连续 1 的个数？将连续为 1 的数段累加后与当前出现最大的连续 1 的个数进行比较，如果超过则更新出现最大的连续 1 的个数，如果没超过继续遍历数组，如果为 0 则累加的值作废；
  • 算法分析：只需要一次遍历，时间复杂度：O（n），使用常数空间，空间复杂度：O（1）。
