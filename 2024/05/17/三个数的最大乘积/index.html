三个数的最大乘积

发表于 2024-05-17 更新于 2024-05-24 分类于 数据结构与算法 ， leetcode

原题

  • 三个数的最大乘积

Code

1.排序

方法与思路：需要得到三个数乘积的最大值，需要考虑到数组元素的正负问题，全部为正数或负数时返回最大三个元素的乘积，有正数有负数时，需要计算两个最小数与一个最大数的乘积和三个最大数的乘积，最后返回较大值即可；

复杂度分析：

  • 时间复杂度：O(nlogn)，n 为数组的长度
  • 空间复杂度：O(1)
class Solution {
public:
    int maximumProduct(vector<int>& nums) {
        size_t length = nums.size();
        // 排序
        sort(nums.begin(), nums.end());
        // 计算两个最小数与一个最大数的乘积
        int product1 = nums[0] * nums[1] * nums[length - 1];
        // 计算三个最大数的乘积
        int product2 = nums[length - 1] * nums[length - 2] * nums[length - 3];
        // 全部为正或负的情况下，一定返回的是product2，不用单独考虑
        // 返回两者中的较大值
        return max(product1, product2);
    }
};
