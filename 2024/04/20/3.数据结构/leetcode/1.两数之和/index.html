两数之和

发表于 2024-04-20 更新于 2024-05-15 分类于 数据结构与算法 ， leetcode

原题

  • 两数之和

Code

1.枚举

思路与方法：遍历数组，按顺序枚举出两个不同的值，然后求和判断是否等于目标值。

复杂度分析：

  • 时间复杂度：O（n^2）
  • 空间复杂度：O（1）

2.哈希+条件查询

思路与方法：构建一个哈希表，元素值为键，元素下标为值；开始遍历数组中的元素并在哈希表中进行条件查询，如果查询到该元素在哈希表中有伙伴，则将这两个元素的下标返回，如果查询不到该元素的伙伴则将该元素插入到哈希表中供后续的元素进行查询。

  • 如果 b = target - a ，则称 b 为 a 的伙伴

复杂度分析：

  • 时间复杂度：O（n）
  • 空间复杂度：O（n）
