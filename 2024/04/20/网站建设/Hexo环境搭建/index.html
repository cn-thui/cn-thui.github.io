1. 安装 Node.js 和 Git

首先，确保你的计算机上安装了 Node.js 和 Git。你可以在它们的官方网站上找到适合你操作系统的安装程序，并按照说明进行安装。

  • Node.js 官方网站
  • Git 官方网站

2.创建 GitHub Page

首先，登录你的 GitHub 账号，然后点击页面右上角的 “New” 按钮来创建一个新的仓库。仓库名为 username.github.io，其中 username 是你的 GitHub 用户名。这样的命名方式可以让你的博客成为 GitHub Pages 的一部分。
